# 题破此生

<figure class="wp-block-image">*</figure>

```
  昨夜睡着睡着鼻子就不通气了，想来今天果然又是感冒了。虽气温较前几日回升了了不少，可外面的风吹到身上仍有一丝寒意，不禁打了个哆嗦。 近来睡眠时长是一再缩减啊，我也不像以前那般每天睡四五个小时就足够了。也不是不能，只是我在尝试还下以前欠的健康债，让这生活中的某些东 西能够尽量稳定在我的控制范围之内，用几份不变去博得万千变化****  机械的洗漱了一番，一天之中只有早上才能有个难得的安静。院子里的不知名树种早已残花褪尽，此刻枝梢上的翠叶在这风中不断的摇摆。两片叶 反复的相遇、相离，有那么一点像众人鼓掌。很显然，我们都为这生命的奇迹欢呼雀跃。可与此同时，关于对未来去处的思考，我们却都藏在心里不 肯对任何人说，因为谁也未曾知晓。突然有那么一句诗冲入脑中，“禅房花木深”，此地虽不是古寺，可我此时怀的是古寺之心，也无碍于到底所处何 地。繁密枝叶，有如三千繁盛生灵，将你我囊括在内的苍生。我们各自“入定”，参往生，悟前道
```

# 意义内外

```
  原本是慢悠悠的咀嚼着昨日的遗章，算了算日子（发现今日似是又要政治学习了）觉得不行，得赶紧去吃早饭，不然会很饿很饿的****  吃什么呢？大概有半个月没有喝粥了，因为没有什么东西乘着，总觉得这粥太过于寡淡。我尚且还离不了人间烟火，因而也就避开了这种饮食，今 天依旧如此吧！想来想去，还是包子就牛奶最合适。“两个梅菜扣肉……”，当我说完这几个字，脑子就突然空了。梅菜扣肉，那不是道菜吗？我可是在 买包子啊！随即瞥了眼菜单，想要确认这已经被认定为错误的错误。结果没有看到，只能礼貌性的走开。我又在远处看了看包子的招牌，梅菜扣肉无疑****  可为什么要叫梅菜扣肉包，百般无解。香菇青菜包，我能理解。可是你这个梅菜扣肉，怎么总觉得有一丝诡异掺杂其中呢？就像那种将一个简单的汉 字看了个把小时之后，突然就不认识那个字的感觉一样。可能失去了其然到其所然之间的转换密匙吧****  还好，走出餐厅后便没有再去想了**
```

# 物致所然

```
  一天的物理课，而且还是两节物理实验****  又要到综合楼去，我也并未抱怨什么，因为不知要抱怨什么****  活动中心楼后的绿化向来不错，只因其以自然生长偏多，每每从这里走都不由得放慢脚步。路两旁的灌木长了一株株浅白的细小花朵，味道不是很浓，往往依靠微风汇集才能闻到花香****  朝花丛中仔细看去，竟有一只只蜂盘绕在花朵之上，不像是人工养殖蜂的模样，也不知是哪的蜂被这些花儿吸引过来。可否告诉我百花如今凋败多少，我好及时去葬花焚香，看着万物尽归所得之中
```

# 天涯里自会相逢

```
  许嵩的新歌中的一句歌词，也许是昨夜一直念叨这局，搞得我此时满脑子都是天涯相逢。我尝试用这解我自己在友人方面的题，却一直困于题中

  众人在方寸之间相识，终将去往天涯？

  可若是有人一味死守故地，不往什么天涯去呢？我们说他被时代抛弃了，可时代——这个人的意志体现不正是我们这群人所推动的吗？所以说到底，是我们彼此在不同的方向抛弃彼此？可我自始至终并未有这种想法呀，集体剥夺了我这个个体的思想？时代是相近意思的概括，或者说是磨去锐利边缘的中心

  立论错误，导致推论错误

  我想表达的还是很模糊，天涯里自会相逢

  自会相逢
```
